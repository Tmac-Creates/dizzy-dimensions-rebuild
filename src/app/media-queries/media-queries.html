Media Queries & Responsiveness Design

What are Media Queries

Media Queries are a feature in CSS that allow a website to change it's layout and styling based on the size of the screen or device.

They are used to make websites responsive, meaning the site can adjust and look good on:

  • Phones 📱
  • Tablets 📲
  • Laptops/Desktops 💻

Note: List should not be places inside paragraph tags. Always close the paragraph before starting any lists.

Why Media Queries are Important?

Whitout media queries:

  • Content can become too large or small on different screens.
  • Lauouts can break or overlap.
  • Spacing can fell wrong on larger screens.

Media queries solve this by applying different styles depending on the screen size.

Basic Syntax

A media query looks like this:

  • @media -> tells CSS to apply rules under certain conditions.
  • (max-width: 600px) -> applies when the screen is 600 pixels or smaller.

How CSS Works with Media Queries

CSS applies styles in layers:

  1. Base styles apply to all screen sizes.
  2. Media queries override those styles when conditions are met:
  • The image uses the base styles by default.
  • On smaller screens, the media query overrides the max-width.

When to Use Media Queries

Media queries should not be added automatically for every screen size, but only when a layout or design issue appears.
Example:

  • Text to large or small
  • Layout is to cramped. (row elements overlap or break.)
  • Elements overlap or break
  • Spacing looks incorrect
