Media Queries & Responsiveness Design

What are Media Queries

Media Queries are a feature in CSS that allow a website to change its layout and styling based on the size of the screen or device.

They are used to make websites responsive, meaning the site can adjust and look good on:

  • Phones 📱
  • Tablets 📲
  • Laptops/Desktops 💻

Why Media Queries are Important?

Without media queries:

  • Content can become too large or small on different screens.
  • Layouts can break or overlap.
  • Spacing can feel wrong on larger screens.

Media queries solve this by applying different styles depending on the screen size.

Basic Syntax

A media query looks like this:

  • @media -> tells CSS to apply rules under certain conditions.
  • (max-width: 600px) -> applies when the screen is 600 pixels or smaller.

How CSS Works with Media Queries

CSS applies styles in layers:

  1. Base styles apply to all screen sizes.
  2. Media queries override those styles when conditions are met:
  • The image uses the base styles by default.
  • On smaller screens, the media query overrides the max-width.

When to Use Media Queries

Media queries should not be added automatically for every screen size, but only when a layout or design issue appears.
Example:

  • Text too large or small
  • Layout is too cramped. (row elements overlap or break.)
  • Elements overlap or break
  • Spacing looks incorrect

They are not required for every screen size - only when needed.

Examples of Responsive Changes

Media queries can be used to:

  • Reduce font sizes on smaller screens.
  • Adjust padding and spacing.
  • Change layout direction (row -> column).
  • Allow items to wrap instead of being forced into one line.
  • Resize images to fit smaller screens.

Flexbox and Responsiveness

Flexbox works well with media queries because it allows layouts to adjust easily.

Common properties used:

  • display: flex
  • flex-direction
  • justify-content

Key Concepts Learned

  • Media Queries make a site responsive.
  • Base CSS applies first, then media queries override when needed.
  • Not every screen size needs its own media query
  • Fix issues only when they appear.
  • Flexbox helps layouts adapt across screen sizes.

Testing Responsiveness

To test responsive design:

  • Use browser developer tools.
  • Switch between device sizes (phone, tablet, laptop/desktop).
  • Resize the browser window.

This helps identify layout issues and decide where media queries are needed.

Summary

Media queries allow a website to adapt to different screen sizes by applying specific styles when needed.
They are essential for creating a modern, responsive design that works across all devices.
